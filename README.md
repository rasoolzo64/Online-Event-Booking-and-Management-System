## Features

### Multi-Role System
- **End Users**: Browse events, book tickets, view booking history
- **Organizers**: Create events, manage bookings, view analytics
- **Administrators**: Approve events, system oversight, user management

### Core Functionality
-  **Secure Authentication** with session-based JWT tokens
-  **Event Management** with admin approval workflow
-  **Real-time Booking** with seat availability tracking
-  **Responsive Design** for all devices
-  **Role-Based Access Control** (RBAC)

## Tech Stack

### Frontend
- **HTML5** - Semantic structure
- **CSS3** - Responsive styling with Flexbox/Grid
- **Vanilla JavaScript** - Client-side interactivity
- **Font Awesome** - Icons

### Backend
- **Node.js** - Runtime environment
- **Express-session** - Session management
- **bcryptjs** - Password hashing
- **mysql2** - Database driver

### Database
- **MySQL** - Relational database management

- **Git/GitHub** - Version control
